# Physical AI and Humanoid Robotics Course

Welcome to the Physical AI and Humanoid Robotics Course repository. This comprehensive course covers the fundamental concepts and practical implementations of embodied artificial intelligence and humanoid robotics.

## Course Structure
The course is organized into 14 comprehensive chapters:

1. Introduction to Physical AI and Humanoid Robotics
2. ROS2 Fundamentals for Robotics
3. Gazebo Simulation Environment
4. Unity for Robotics Simulation
5. NVIDIA Isaac Platform
6. Vision-Language-Action Models
7. Hardware Components and Integration
8. Laboratory Exercises Part 1
9. Edge AI and Computing Platforms
10. Perception Systems
11. Control Systems
12. Learning and Adaptation
13. Ethics and Safety in Robotics
14. Future Directions and Research

## Chatbot Assistant
This course includes an intelligent chatbot assistant that helps students ask questions about the course content:

- **Floating Widget**: Available on all pages as a floating chat interface
- **Dedicated Page**: Accessible at `/chatbot` for focused interaction
- **Knowledge Base**: Comprehensive coverage of all 14 course chapters
- **GitHub Integration**: Direct link to this repository within the chatbot interface
- **Smart Responses**: Keyword-based system that provides relevant information about robotics topics

## Getting Started

To run this documentation site locally:

1. Clone the repository
2. Install dependencies: `npm install` or `yarn install`
3. Start the development server: `npm start` or `yarn start`
4. Open your browser to http://localhost:3000
